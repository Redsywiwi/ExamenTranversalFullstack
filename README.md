**DEPENDENCIAS**

-laragon
-jdk 21
-java
---------------

**Como correr**

-Se debe abrir laragon y iniciar los servicios de apache. my sql y mmailpit
-despues dentro de la seccion de consultas se debe crear la base de datos "db_hospital_vm"
-una vez la base de datos este creada se va al proyecto y le da a correr

---------------

**que hace?**

-es un sistema de base de datos. solamente la parte de backend
-permite guardar datos de usuarios, detalles y otros en una base de datos local
